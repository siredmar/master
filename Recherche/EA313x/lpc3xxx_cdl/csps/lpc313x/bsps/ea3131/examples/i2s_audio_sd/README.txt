1. LPC313x Audio Demo application using SD card as storage:
===========================================================
This application will demonstrates the following functionality:
a. Playback audio wave files (Raw PCM wave files) stored in 
   SD card
b. Record audio from Microphone connected to Mic socket of 
   EA3131 & stored as Wave file (Raw PCM Wave file) into SD card

The recording Wave file will have:
44100KHz sample rate, stereo samples, 16 bits per sample

2. Supported Targets:
=====================
The following target builds are supported for this application.
a. ISRAM 
b. ISRAM with cache enabled
c. SDRAM
d. SDRAM with cache enabled
